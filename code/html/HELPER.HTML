Checker URL to more secure

VirusTotal

VirusTotal is a widely used online tool that analyzes suspicious files
and URLs to detect malware and other security threats. It aggregates
results from multiple antivirus engines and other security tools to
provide comprehensive insights into the safety of a URL. Users can simply
paste the URL they want to check into the VirusTotal website,
and it will scan the link for any potential threats.

Google Safe Browsing

Google Safe Browsing is a service provided by Google that helps protect users
from visiting dangerous or deceptive websites. It maintains a constantly
updated list of known unsafe websites and warns users before they visit
these sites. Google Safe Browsing is integrated into various browsers,
but you can also manually check a URL by using the Google Safe Browsing website.
Just enter the URL, and it will inform you if the website is considered safe or not.

Use the Authenticator to more secure

Google Authenticator

Google Authenticator is a widely used authenticator app developed by Google.
It generates one-time passcodes for two-factor authentication (2FA) to add an
extra layer of security to your accounts. Google Authenticator is available
for both Android and iOS smartphones, and you can also use it on your PC
via browser extensions like Authenticator Plus.
